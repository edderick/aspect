COMP6035

Bounded Integer Division

A Tale of Woe -- By Edward Seabrook

The second coursework for Aspect Oriented Programming requires us to use C++ Template Meta-programming, to build a set of templates that allow their user to write expressions that are evaluated at runtime. For example, the expression:

x + (x-2) * (x-3)
could be written as:
ADD<X, MULT<SUB<X, LIT<2> >, SUB<X, LIT<3> > > > Expr;

The second question of the coursework requires this to be extended to offer bounds on the inputs.

0 ≤ x ≤ 5
is written as:
X<BOUNDS<0,5> > x;

The aim of this is to allow bounds to be created over the whole expression, the example given is for addition and multiplication:

Consider a sample expression with a variable that carries these bounds:

(x_{0,5} + 3) * ( x_{0,5} + 5)

Then we can calculate the bounds on the whole expression above as follows:

3 ≤ x_{0,5} + 3 ≤ 8 and 5 ≤ x_{0,5} + 5 ≤ 10

Consequently we have that

15 ≤ ( x_{0,5} + 3 ) * ( x_{0,5} + 5 ) ≤ 80

This example is trivial to implement. The bounds for each subexpression are simply passed to their parents, and the bounds for the expressions are calculated. This also holds for subtraction.
